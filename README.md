# System Delta - Advanced Logic Simulation (Inform 7)

Ovaj projekt predstavlja implementaciju interaktivne simulacije sustava unutar deklarativne paradigme koristeći jezik **Inform 7**.

## Ključne značajke
- **NLP sučelje:** Procesiranje naredbi prirodnog jezika.
- **Termodinamički model:** Simulacija rasta i pada temperature servera ovisno o stanju generatora i hlađenja.
- **Sustav održavanja života:** Vremenski ovisna simulacija razine kisika.
- **Logička verifikacija:** Uvjeti pobjede temeljeni na konjunkciji predikata (Clearance AND Power AND Stable Temperature).

## Kako pokrenuti
1. Kopirajte sadržaj `Source/SystemDelta.ni` u [Borogrove.app](https://borogrove.app) ili lokalni Inform 7 IDE.
2. Pritisnite **Go** ili **Compile**.
